Plugin Overview

AdvancedServerList

AdvancedServerList enhances the server list functionality, providing additional features and customisation options.

ajLeaderboards

ajLeaderboards adds leaderboard functionality to your server, allowing players to compete and track their progress.

BedWars

BedWars is a popular minigame plugin that simulates the Bed Wars game mode, where players protect their beds while attempting to destroy the beds of other teams.

ChestShop

ChestShop enables players to create their own shops by placing signs on chests, providing a convenient way for players to buy and sell items with each other.

CMI

CMI, or Customizable Minecraft Interface, is a comprehensive plugin that adds many features and enhancements to the Minecraft server, including commands, utilities, and more.

CMILib

CMILib is a library plugin that provides essential functionality and utilities for other plugins to utilise.

CoreProtect

CoreProtect is a powerful block logging and rollback plugin, allowing server administrators to track changes made by players and revert them if necessary.

CrashTP

CrashTP is a plugin designed to mitigate the effects of server crashes by automatically recovering players to the world spawn in the event of a corrupt chunk or server crash. This ensures that players are safely relocated to a stable area, minimising disruption and potential loss of progress caused by unexpected server issues.

CrisTreeCapitator

CrisTreeCapitator is a plugin that simplifies the process of cutting down trees by allowing players to chop down the entire tree by breaking just one block of its trunk.

DeluxeMenus

DeluxeMenus enables server owners to create custom menus and GUIs, providing a user-friendly interface for accessing server features and commands.

DiamondBank

DiamondBank is a plugin that adds banking functionality to the server, allowing players to deposit and withdraw virtual currency (diamonds) from their accounts.

DiscordSRV

DiscordSRV is a bridge plugin that integrates Minecraft chat with Discord, allowing players to communicate across both platforms.

Dynmap

Dynmap generates dynamic, interactive maps of the Minecraft world, which can be viewed through a web browser, providing players with a visual representation of the server.

Graves

Graves is a plugin that creates grave markers at the location where players die, allowing them to retrieve their items upon respawning.

ImageOnMap

ImageOnMap allows players to display custom images as maps within the game world, adding a decorative and personal touch to their builds.

InteractiveChat

InteractiveChat enhances the in-game chat experience by adding features such as clickable links, hover tooltips, and more.

InteractiveChatDiscordSrvAddon

InteractiveChatDiscordSrvAddon is an addon for InteractiveChat that integrates Discord messages into the in-game chat, allowing players to communicate seamlessly across both platforms.

InvisibleFrames

InvisibleFrames is a utility plugin that allows server administrators to create invisible frames within the game world, which can be used for various purposes such as hiding command blocks or creating hidden passages.

IridiumSkyblock

IridiumSkyblock is a plugin that allows players to create and manage their own skyblock islands, providing a unique and challenging gameplay experience.

LiveKit

LiveKit is a staff plugin designed to enhance server management by allowing staff members to interact with the server remotely from their mobile devices. With LiveKit, staff can efficiently monitor and manage various aspects of the server, including player activities, chat moderation, and server settings, providing greater flexibility and responsiveness to server management tasks.

LuckPerms

LuckPerms is a permissions management plugin that provides a flexible and powerful system for managing permissions and groups on the server.

Multiverse-Core

Multiverse-Core is a core plugin for managing multiple worlds on the server, allowing players to easily teleport between different dimensions and worlds.

Multiverse-Inventories

Multiverse-Inventories is an addon for Multiverse-Core that enables separate inventories for each world, allowing players to have different sets of items and resources in different dimensions.

Multiverse-Portals

Multiverse-Portals is an addon for Multiverse-Core that adds portal functionality, allowing players to create custom portals for teleporting between worlds.

PlaceholderAPI

PlaceholderAPI is a plugin that provides placeholders for various data and information, allowing other plugins to display dynamic content in messages, signs, and other places.

Plot2Dynmap

Plot2Dynmap is a plugin that integrates PlotSquared plots with Dynmap, allowing players to view and interact with plots on the dynamic map.

PlotManagement

PlotManagement provides tools and commands for managing and protecting plots on the server, allowing players to claim and customize their own plots.

PlotSquared

PlotSquared is a plugin that allows server owners to create and manage plots, providing players with their own customizable building spaces.

ProtocolLib

ProtocolLib is a library plugin that allows developers to modify the Minecraft protocol, enabling the creation of custom features and enhancements.

SMP-API

SMP-API is a plugin that provides an API (Application Programming Interface) for server developers, allowing them to create custom plugins and addons for the server.

Vanillahungergames

Vanillahungergames is a plugin that adds Hunger Games-style gameplay to the server, where players compete to be the last one standing in a battle royale arena.

VentureChat

VentureChat is a plugin that enhances the chat system on the server, providing features such as channels, private messaging, and chat formatting.

ViaBackwards

ViaBackwards is a plugin that enables compatibility with older versions of Minecraft, allowing players using older clients to connect to the server.

ViaVersion

ViaVersion is a plugin that enables compatibility with newer versions of Minecraft, allowing players using newer clients to connect to the server.

VillagerInfo

VillagerInfo is a plugin that enhances the functionality of staff to interact with villagers on the server, providing additional information about their behaviour.

WildRTP

WildRTP is a plugin that adds a random teleportation feature to the server, allowing players to teleport to random locations within the game world.

WorldEdit

WorldEdit is a powerful in-game map editor that allows players to quickly and efficiently manipulate the terrain and structures within the game world.
